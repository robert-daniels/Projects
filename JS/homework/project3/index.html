  2. Why is the JavaScript textContent property safer than innerHTML?

    The textContent option is safer than innerHTML as it treats what is passed to it as a literal string. The innerHTML element treats what is passed to it as actual HTML. The latter may, in theory, render. This rendering, especially if passed from an end user, may alter the webpage is unanticipated / undesired ways.

  4. Why do we rarely, if ever use document.write() ?

    The call to document.write(str) writes HTML directly into the page as it's called. The consequence is that this usually only works as intended while the page is loading. If this is called after the page has loaded, the existing document is erased, leaving the page at whatever has been passed to document.write().

  5. What is a JavaScript method? Define and provide two examples.

    Methods (functions) are actions that can be called to get the script to "do something" to "something else." They can return nothing (void) or return something like a Boolean True/False. Methods are often used to manipulate various properties of objects.

    Some examples of JavaScript method examples are as follows:

    • document.createElement(tag) : Creates an element with the tag
    • node.append(elem) : Takes the parameter and inserts it into the specified node, at the end.
     
  6. What is a JavaScript property? Define and provide two examples.

    A property in JavaScript is a value descriptor of the object that it is associated with. It can be called upon as a value, or manipulated with methods.

    Some examples of JavaScript properties are as follows:

    • className : the name of the class the object is associated with
    • hidden : a Boolean true / false if the object is currently hidden.

  8. What is a CSS selector? Define and provide two examples.

    A CSS selector is a call in a stylesheet or other style source to an html object that matches the header of the call. Examples include an element selector such as p or h1, and an id selector such as #container.

  9. What is a CSS property? Define and provide two examples.

    Given a CSS selector, a CSS property will cause that style to be set on the item(s) selected. For example, calling #container selector with font-weight = bold will bold all text within an element with the id ="container". This is in somewhat contrast to the syntax for the JavaScript equivalent fontWeight property.
